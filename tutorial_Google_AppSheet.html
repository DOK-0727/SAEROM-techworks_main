Data Views Actions Security

Data

1) 좌측 "Data" 버튼 클릭
2) 좌측 좌측 "+" 버튼 클릭
3) datasource 추가
4) Data 설정

  • Data
    • NAME: 데이터 이름 설정
    • TYPE: 데이터 유형 설정
    • KEY?: 고유 식별자 선택
    • LABEL?: 대표 표시자 선택
    • FORMULA: 계산 수식 설정
    • SHOW?: 화면 표시 여부 선텍
    • EDITABLE?: 수정 가능 여부 선택
    • REQUIRE?: 필수 입력 여부 선택
    • INITIAL VALUE: 기본값 설정
    • DISPLAY NAME: 화면 표시 이름 설정
    • DESCRIPTION: 데이터 설명 설정
    • SEARCH?: 검색 가능 여부 선택
    • SCAN?: 스캔 사용 여부 선택
    • NFC?: NFC 사용 여부 선택
    • PII?: 개인정보 여부 선택

Views

1) 좌측 "Views" 버튼 클릭
2) 좌측 "+" 버튼 클릭
3) View 생성
4) View 설정

  • Views
    • PRIMARY NAVIGATION: 하단 메뉴 뷰 추가
    • MENU NAVIGATION: 사이드 메뉴 뷰 추가
    • REFERENCE VIEWS: 참조용 뷰 추가
  • View
    • View name: 뷰 이름 설정
    • For this data: 연결된 테이블 설정
    • View type: 뷰 디자인 설정
    • Position: 뷰 위치 설정
  • View Options
  • Display
    • Icon: 표시 아이콘 설정
    • Display name: 표시 이름 설정
    • Show if: 표시 조건 설정
  • Behavior
    • Event Actions: 특정 이벤트에 따라 실행되는 액션 설정
    • App link: 뷰 링크
  • Documentation
    • Descriptive comment: 뷰 설명 설정

Actions

1) 좌측 "Actions" 버튼 클릭
2) 좌측 "+" 버튼 클록
3) Action 생성
4) Action 설정

  • Action
    • Action name: 액션 이름 설정
    • For a record of this table: 대상 테이블 설정
    • Do this: 실행 동작 설정
    • Set these columns: 데이터 계산 수식 설정
    • Position: 액션 위치 설정
  • Display
    • Display name: 표시 이름 설정
    • Action icon: 표시 아이콘 설정
  • Behavior
    • Only if this condition is true: 실행 조건 설정
    • Needs confirmation?: 사용자 확인 여부 선택
    • Confirmation Message: 사용자 확인 메시지 설정
  • Advanced
    • Inputs: 사용자 입력값 설정
  • Documentation
    • Descriptive comment: 액션 설명 설정

Security

1) 좌측 "Security" 버튼 클릭
2) table 선택
3) Security 설정

  • Security filters
    • Security filter: 필터 조건 설정
    • Access mode: 필터 대상 선택
