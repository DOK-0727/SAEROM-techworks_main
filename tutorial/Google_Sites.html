Google Sites 튜토리얼

삽입 페이지 테마 미리보기 링크 공유 설정 버전 기록 게시

삽입

1) 우측 "삽입" 버튼 클릭
2) 삽입 요소 추가

  • 삽입 요소
    • 텍스트 상자: 텍스트 추가
    • 이미지: 이미지 추가
    • 삽입: URL또는 HTML코드 추가
    • Drive: Google Drive의 요소 추가
    • 콘텐츠 블록: 사진&텍스트 추가
    • 접을 수 있는 그룹: 접을 수 있는 텍스트 추가
    • 목차: 제목 타입 텍스트의 내비게이션 추가
    • 이미지 캐러셀: 이미지 슬라이드 추가
    • 버튼: 링크 버튼 추가
    • 구분선: 선 추가
    • 공백: 빈 공간 추가
    • 소셜 링크: 링크 이미지 추가
    • 자리표시자: 업로드 요소 추가
    • Cloud Search: Google Workspace 검색창 추가
    • YouTube: YouTube 요소 추가
    • Calendar: Google Calendar 요소 추가
    • 지도: Google Maps 요소 추가
    • Docs: Googlde Docs 요소 추가
    • Slides: Google Slides 요소 추가
    • Sheets: Google sheets 요소 추가
    • 설문지: Google form 요소 추가
    • 차트: Google chart 요소 추가

페이지

1) 우측 "페이지" 버튼 클릭
2) 우측 하단 "+" 버튼 호버
3) 페이지 추가

  • 페이지
    • 새로운 메뉴 섹션: 내비게이션 빈 섹션 추가
    • 전체 페이지 삽입: URL 또는 HTML 코드 전체 페이지 추가
    • 새 링크: 내비게이션 링크 이동 섹션 추가
    • 새 페이지: 기본 페이지 추가

테마

1) 우측 "테마" 버튼 클릭
2) 테마 설정

  • 색상
    • 기본: 헤더 배경 색 설정
    • 배경: 사이트 배경 색 설정
    • 제목: 제목, 부제목 색 설정
    • 본문 텍스트: 일반 텍스트, 작은 텍스트 색 설정
  • 텍스트
    • 텍스트: 텍스트 타입 선택
    • 글꼴: 글자 폰트 설정
    • 크기: 글자 크기 설정
    • 서식: 글자 서식 설정
    • 정렬: 글자 정렬 설정
    • 줄 간격: 글자 세로 간격 설정
    • 단락 간격: 글자 가로 간격 설정
  • 이미지
    • 머리글: 헤더 배경 이미지 설정
    • 로고: 로고 이미지 설정
    • 파비콘: 탭 이미지 설정
  • 탐색
    • 모드: 내비게이션 위치 설정
    • 배경: 내비게이션 배경 색 설정
    • 선택한 페이지: 내비게이션 두께 설정
  • 컴포넌트
    • 버튼: 링크 버튼 색 설정
    • 구분선: 구분선 디자인 설정
    • 링크: 링크 텍스트 색 설정
    • 이미지 캐러셀: 이미지 캐러셀 점 색 설정
  • 간격
    • 밀도: 요소 사이 간격 설정
    • 사이트 너비: 사이트 너비 설정
    • 프레임 표시: 사이트 프레임 설정

미리보기

1) 우측 상단 "미리보기" 버튼 클릭
2) 미리보기 화면 확인

  • 화면
    • 대형 화면: 컴퓨터 화면 비율
    • 태블릿: 태블릿 화면 비율
    • 전화번호: 핸드폰 화면 비율

링크

1) 우측 상단 "링크" 버튼 클릭
2) 링크 복사(https://sites.google.com/saerom.hs.kr/입력값)

공유

1) 우측 상단 "공유" 버튼 클릭
2) Gmail 입력
3) 권한 설정
4) 우측 하단 "완료" 버튼 클릭

  • 일반 액세스
    • 초안: 편집 화면 권한 설정
    • 게시된 사이트: 적용 화면 권한 설정

설정

1) 우측 상단 "설정" 버튼 클릭
2) 사이트 설정

  • 탐색
    • 모드: 내비게이션 위치 설정
    • 검색 설정: 내비게이션 검색 설정
  • 브랜드 이미지
    • 로고: 로고 이미지 설정
    • 파비콘: 탭 이미지 설정
  • 뷰어 도구
    • 정보 아이콘: 사이트 업데이트 시간, 문의 설정
    • 앵커 링크: 제목 타입 텍스트 복사 설정
  • 분석: 사용자 통계 설정
  • 공지사항 배너
    • 배너 표시: 배너 표시 설정
    • 배너 색상: 배너 색상 설정
    • 공지: 배너 내용 설정

버전 기록

1) 우측 상단 "⋮" 버튼 클릭
2) 우측 상단 "버전 기록" 버튼 클릭
3) 우측 버전 기록 확인

게시

1) 우측 상단 "게시" 버튼 클릭
2) 웹 주소 입력
3) 우측 하단 "게시" 버튼 클릭
