프로젝트 기록

1) 좌측 "프로젝트 기록" 버튼 클릭
2) 우측 프로젝트 기록 확인

트리거

1) 좌측 "트리거" 버튼 클릭
2) 우측 하단 "트리거 추가" 버튼 클릭
3) 실행할 함수 선택
4) 이벤트 소스 선택
5) 우측 하단 "저장" 버튼 클릭

  • 이벤트 소스
    • 시간 기반: 시간에 따라 특정 함수 작동
    • 캘린더: Google 캘린더의 이벤트에 따라 특정 함수 작동

배포

1) 우측 상단 "배포" 버튼 클릭
2) 배포 유형 선택
3) 배포 권한 설정
4) 우측 하단 "배포" 버튼 클릭
5) URL 복사(https://script.google.com/a/macros/saerom.hs.kr/s/Apps Script ID/dev?pli=1&authuser=0

  • 배포 유형
    • 웹 앱: 웹 사이트
    • API 실행 파일: API
    • 부가기능: 확장 프로그램
    • 라이브러리: 라이브러리

공유

1) 우측 상단 "공유" 버튼 클릭
2) Gmail 입력
3) 권한 설정
4) 우측 하단 "완료" 버튼 클릭

삭제

1) 우측 상단 "배포" 버튼 클릭
2) 아래 "배포 관리" 버튼 클릭
3) 삭제할 프로젝트 선택
4) 우측 상단 "보관" 버튼 클릭
5) 우측 하단 "보관처리" 버튼 클릭
6) 좌측 "프로젝트 기록" 버튼 클릭
7) 우측 하단 "삭제" 버튼 클릭
8) 삭제할 프로젝트 선택
9) 우측 하단 "삭제" 버튼 클릭
