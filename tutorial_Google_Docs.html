데이터 베이스 프로젝트 문서

데이터 베이스

1)Docs 생성
2) 공유 설정 "링크가 있는 모든 사용자"로 변경
3) 링크 복사(https://docs.google.com/spreadsheets/d/Sheets ID/edit?usp=sharing)
4) Docs 확장 프로그램에서 Apps Script 생성
5) 코드 에디터 JavaScript 코드 Apps Script에 붙여넣기
6) Apps Script 배포 7) URL 복사(https://script.google.com/a/macros/saerom.hs.kr/s/Apps Script ID/exec)
8) 코드 에디터 HTML 코드 붙여넣기
9) Docs ID, Apps Script URL 코드에 붙여넣기

프로젝트 문서

1) 프로젝트 문서 생성
2) 문서 이름 프로젝트명으로 변경
3) 프로젝트명 탭 이름 프로젝트명으로 변경
4) 프로젝트명 탭에 표 작성
5) 코드(프론트엔드), 코드(백엔드) 탭에 코드 붙여넣기
6) 개발일지 탭에 개발일지 작성
